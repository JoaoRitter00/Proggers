Aula sobre git

Oque é git

Basicamente, git é um sistema de controle de versões, mas é mais usado para registrar o historico de edições de um arquivo. O termo "Git", uma gíria em inglês britânico para cabeça dura

Como usar

  1. Primeiramente, baixamos o conector git para usar os comandos no terminal

  2. agora que temos os "drivers" do git, criamos uma conta no github, algo como um portfolio, usado pra registrar seus arquivos.

  3. agora somente criar um repositorio no github

  4. agora podemos usar os comandos no terminal, vou listar os comandos usados na aula de hoje e seus funcionamentos

    • cd [pasta]

      • comando basico do windows, usado pra se movimentar entre as pastas e arquivos

    • ls

      • lista os arquivos da pasta ou diretorio.

    • git config --global credential.helper manager-core

      • serve para forçar o git salvar suas informações para não ter que colocar tudos denovo.

    • git clone [link do diretorio]

      • esse você usa pra declarar o link do seu repositorio do git hub

    • git add .

      • git add [filename] seleciona um arquivo, e move ele pra determinada area, marcando-o para inclusão no próximo commit.

    • git commit -m "arquivo"

      • São criados unidades estruturais de um cronograma com o comando git commit para capturar o estado de um projeto naquele momento.

    • git push

      • O comando git push é usado para enviar o conteúdo do repositório local para um repositório remoto. (github no caso)

      os comandos git push, git add e git commit[...], é usado pra salvar e mandar pro diretorio no github
